# Quipper API benchmark Haskell version

# How to install Haskell

- Stable way: [Haskell Platform](https://www.haskell.org/platform/)
- I want Latest GHC:
  - Haskell for Mac OS X http://ghcformacosx.github.io/
  - `$ brew install ghc --edge`
  - Compile from source

# How to build

```
$ cabal sandbox init
$ cabal install
$ cabal configure
$ cabal build
```

# How to run

```
$ ./run.sh
```
